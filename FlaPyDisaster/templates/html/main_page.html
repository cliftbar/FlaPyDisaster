Parametric hurricane model

FlaPy Disaster implements the NOAA NWS23 hurricane model to calculate and display hurricane wind footprints. The UI is built usingPython/Flask, calculations are driven by an Scala/Akka server, and mapping is powered by Leaflet.js. Data can be imported from NOAA, Unisys, and user data files.
